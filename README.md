# Tangle

A tool for exporting markdown code blocks to files.

## Requirements

- [Python 3.11.0](https://www.python.org/downloads/)

## Installation

Clone the repository, and use `pip` to install the project as a python package.

```bash
pip3 install .
```

## Usage

```bash
tangle <file>
```
